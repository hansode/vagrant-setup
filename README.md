vagrant-setup
=============

Requirements
------------

+ [bash](http://www.gnu.org/software/bash/)
+ [make](http://www.gnu.org/software/make/)

Installation
------------

```
$ git clone https://github.com/hansode/vagrant-setup.git
```

Usage
-----

```
$ vagrant-setup <box-name>
```

Getting Started
---------------

```
$ cd /path/to/dir
$ vagrant-setup testing-box
```
